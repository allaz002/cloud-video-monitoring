# cloud-video-monitoring frontend

Angular dashboard for `cloud-video-monitoring`.

The dashboard expects the backend API to run at:

```text
http://localhost:8081
```

## Install

Install dependencies if needed:

```powershell
npm.cmd install
```

## Run

Start the development server:

```powershell
npm.cmd start
```

Open:

```text
http://localhost:4200
```

## Build

```powershell
npm.cmd run build
```

No global Angular CLI installation is required for normal project commands.
