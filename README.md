# Cloud Video Monitoring

Eine cloudnahe Video-Monitoring-Plattform.

## Ziel

Mehrere simulierte Kameras senden Events an ein Spring-Boot-Backend.  
Das Backend speichert Kamera- und Eventdaten, berechnet Statusinformationen und stellt REST APIs bereit.  
Ein Angular-Dashboard zeigt Kameras, Status, Events und Alarme.

## Geplanter Tech Stack

- Java 21
- Spring Boot
- PostgreSQL
- Angular
- Python Camera Simulator
- Docker Compose
- GitHub Actions
- Spring Boot Actuator
- Azure-Mapping
